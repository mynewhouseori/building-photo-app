# צילום מתועד לבניין

## Short Description
אפליקציה לתיעוד תמונות באתר בנייה עם פרטי אתר, צלם, תאריך, שעה ומיקום.

## Full Description
צילום מתועד לבניין מאפשרת לתעד תמונות מהשטח בצורה מסודרת ואמינה. מזינים פרטי אתר וצלם, מצלמים, מוסיפים הערות, ובוחרים האם לצרף תאריך, שעה ומיקום. התוצאה היא תמונה מתועדת שניתן לשמור ולשתף.

מתאים למפקחי בנייה, קבלנים, מנהלי פרויקטים, יזמים ודיירים שרוצים תיעוד ברור של התקדמות, ליקויים, מסירות ובדיקות שטח.
